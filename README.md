## Skills

### [cognitive-load-minimization](cognitive-load-minimization/)

[![skills.sh](https://skills.sh/b/EgehanYildiz/skills)](https://skills.sh/EgehanYildiz/skills/cognitive-load-minimization)

Audit and reduce the **cognitive load** of any product surface — landing page, SaaS dashboard, onboarding flow, signup form, navigation, pricing page, or marketing copy. Fans out four parallel expert lenses (IA, visual hierarchy, content, interaction), then adversarially verifies every finding so plausible-but-wrong ones don't survive. Every recommendation cites a named law (Hick, von Restorff, Gestalt, Fitts, Tesler…), grounded in Sweller's Cognitive Load Theory.

```bash
npx skills add EgehanYildiz/skills --skill cognitive-load-minimization
```
